## Why This Matters

A competition project is not a flat list of models. Each route has a role:

- `mainline`: current best path to leaderboard.
- `source`: useful evidence or teacher, not necessarily submitted.
- `control`: helps compare difficulty and mechanism.
- `candidate`: needs a spec/gate before real execution.
- `hold`: useful but not active.
- `reject`: should not continue or package without a new reason.
